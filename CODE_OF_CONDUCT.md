# 行为准则 / Code of Conduct

## 我们的承诺 / Our Pledge

这是一个为劳动者服务的项目。我们承诺让贡献者和用户的体验**没有骚扰**——不论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教、或性向。

This is a project that serves workers. We pledge to make participation **harassment-free** for everyone — regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## 我们的标准 / Our Standards

### 我们鼓励 / We encourage

- 尊重不同的观点和经验 / Respect for differing viewpoints
- 接受建设性批评 / Graceful acceptance of constructive criticism
- 关注对劳动者最好的事 / Focus on what is best for workers
- 对其他社区成员保持同理心 / Empathy towards other community members
- 把"匿名化别人的故事"当作必要而不是负担 / Treat anonymization of others' stories as essential, not a burden

### 我们不接受 / We do not accept

- 任何形式的骚扰 / Harassment of any kind
- 人身攻击或政治攻击 / Personal or political attacks
- 公开/私下骚扰 / Public or private harassment
- 未经许可发布他人的私人信息 / Publishing others' private information without permission
- 推广特定政治立场（左/右都不要）/ Promoting any specific political stance
- 推广特定公司/工会/律所/招聘平台 / Promoting any specific company/union/law firm/recruiter
- 仇恨言论 / Hate speech
- 教唆违法 / Encouraging illegal activity (whistleblowing legally protected acts is fine; stealing data is not)

## 我们的边界 / Our Scope

这个项目不是**任何政治组织**的延伸，不号召任何**集体行动**，不替代**专业法律建议**，不代表**任何特定阶级立场**。

它的核心使命简单：**让单个劳动者在普遍偏向资本的话语环境里多一个站在自己这边的声音**。

任何与这个使命冲突的内容会被移除。

## 强制执行 / Enforcement

如果你看到违反此准则的行为，请通过以下方式联系：

- 在 issue 中标记 maintainer
- 通过 GitHub 私信
- 发邮件到（如果未来设立联系邮箱）

所有报告会被认真和保密地审查。
