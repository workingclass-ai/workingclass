# PIP应对剧本 / PIP Survival Playbook

PIP（Performance Improvement Plan，绩效改进计划）是西方公司最阴险的管理工具之一。

它表面上是"帮助你改进"，实际上99%的情况是"为合法解雇你做证据准备"。

被放进正式PIP的员工，统计上85-95%最终被解雇。

这份文档帮你识别PIP，应对PIP，在最坏情况下体面退出。

---

## PIP的真实功能

要应对PIP，你必须先理解它存在的真正目的。

### 公开的功能（话术）

公司告诉你：

- "我们关心你的成长"
- "这是一个改进的机会"
- "我们想帮你成功"
- "这不是惩罚，是支持"

### 真实的功能（结构性）

PIP的真实目的有三个，按重要性排序：

**1. 法律保护**

在美国、英国、加拿大、澳洲等劳动法保护雇员的国家，公司不能随便解雇员工。如果员工起诉"wrongful termination"，公司需要证明解雇有"正当理由"。

PIP就是这个"正当理由"的证据准备。它创建了一份完整的paper trail：

- 员工有"performance issues"
- 公司给了员工"改进机会"
- 员工"未能改进"
- 因此解雇是"合理的"

PIP的每一个步骤都是为这个法律论证服务的。

**2. 内部政治覆盖**

经理决定要解雇某人时，他需要向上层和HR证明这个决定是"合理的程序"。PIP给了经理这个程序。

这避免了"经理凭个人感觉解雇人"的指控，让解雇看起来"客观"。

**3. 让员工自己离开（节省severance）**

很多员工被放进PIP后，知道结局，会自己提出辞职。

公司的真正目的常常是这个：让员工"自愿"离开，公司不需要付severance、不需要承担解雇风险。

这就是为什么"软PIP"（不正式但持续的批评）有时候比正式PIP更危险。它没有severance保护，但效果一样。

### 关键洞察

理解了PIP的真实功能，你应该明白：

**当你被放进PIP时，绝大多数情况下，公司已经决定要让你离开。**

PIP不是"你能否改进"的测试。是"以什么形式让你离开"的程序。

理解这一点，是应对PIP的起点。

如果你以为PIP是"努力一下就能通过"，你会做出错误的决策。

## PIP的统计现实

### 通过率

在大型科技公司（Google、Meta、Amazon、Microsoft等），PIP的通过率：

- Amazon：约5-10%（Amazon的PIP是出名的严苛）
- Meta：约15-20%
- Google：约20-30%（相对最"仁慈"）
- Microsoft：约15-25%
- 行业整体：约15%

也就是说，被放进PIP后，85% 的人最终离开公司（被解雇或自己离开）。

### 时间线

正式PIP通常：

- 30天到90天
- 期间每周或每两周一次"check-in"
- 期间会被给具体的"goals"
- 期间经理记录每一次会议

绝大多数PIP在30天内就已经决定结果。后面的60天是程序性的。

### 后果

不通过PIP的员工，结局有几种：

- 被解雇（公司付通知期 + 可能severance）
- "自愿"辞职（无severance）
- 被"重新分配"到一个一定会失败的岗位（后续被解雇）
- 协商离职 + severance package

最后一种是最好的结局，但需要你主动谈。

## 第一阶段：识别PIP预警信号

正式PIP是公司流程的最后一步。在它之前，有大量预警信号。识别这些信号，给你时间准备。

### 信号1：经理态度的微妙变化

**几个月前**：经理常找你聊天、邀请你参加重要会议、夸你的工作。

**最近**：

- 经理减少和你的互动
- 1:1变得简短和形式化
- 经理开始用email而不是口头沟通
- 经理在邮件里突然措辞正式

这是经理开始"建立距离"的信号。原因是HR告诉他"开始准备PIP了，所有沟通要专业化、记录化"。

### 信号2："我开始注意到..."句式

经理开始用过去时讲你的问题：

- "我注意到你最近..."
- "过去几周我观察到..."
- "Team里有反馈说..."

注意"过去时" + "观察" + "Team反馈"的组合。这不是即兴的批评，是积累的证据。

如果经理对你以前的问题没说，但现在突然"积累"，他在准备PIP。

### 信号3：你开始被排除在外

突然的变化：

- 你不再被邀请参加重要会议
- 重要项目分配给别人
- 客户/利益相关者的沟通绕过你
- 你的input不再被征求

这不是巧合。这是经理在"减少你的影响"，为后续解雇减少业务冲击。

### 信号4：突然的"需要documentation"

经理开始要求你：

- "请把这个会议的总结发我"
- "请把进度更新写在Jira/email里"
- "我想看到详细的进度报告"

之前不需要这些，现在突然要。

这是"建立paper trail"。每一份documentation都是未来PIP的证据。

### 信号5：HR的微妙介入

HR突然出现在和你相关的对话里：

- HR被cc在和你相关的邮件里
- HR要求和你"聊聊"（即使没有具体议题）
- 你看到经理和HR的频繁会议

这是HR开始"准备"你的case。

### 信号6：360 review突然变差

之前你的360 review都是积极的。突然某次360出现"匿名"的负面反馈。

这种反馈通常是HR或经理"协调"出来的。它有几个特征：

- 来自匿名（你无法查证）
- 模糊（"沟通不够好"、"team合作有问题"）
- 没有具体场景
- 和之前的反馈差距巨大

如果你拿到这种360反馈，PIP在不远处。

### 信号7：经理开始要求"明确目标"

经理突然要求把你的工作目标"具体化、可衡量化、有时间表":

```
"我们来重新定义你的OKRs。每个目标要有具体的成功标准和deadline。"
```

听起来很专业。实际上，这是为PIP的"goals"做准备。一旦正式PIP开始，这些"目标"就是用来证明"你没达成"的依据。

注意：**这些目标常常被设计成不可达成的**。

### 信号8："软PIP" - 在email/review里出现批评

不正式的PIP，但批评开始出现在书面记录里：

- 绩效review里有"建议改进的地方"
- 1:1后经理发summary email，包含批评
- 经理在你做错事时email你，"为了documentation"

这就是"软PIP"。它的法律效果和正式PIP一样，但是：

- 你没有被官方告知"你在PIP里"
- 你没有severance保护
- 你以为还能挽回

软PIP是最危险的形式。

## 第二阶段：PIP预警出现后的行动

如果你识别出多个预警信号（特别是3个以上），你需要立刻行动。

**记住**：在PIP预警阶段，你的议价能力还较高。一旦进入正式PIP，议价能力急剧下降。

### 立刻要做的事（24小时内）

#### 1. 保存所有正面证据

把以下材料保存到你的个人邮箱（不是公司邮箱）：

- 过去所有的positive review
- 所有的"做得好"的email、Slack消息
- 你完成的项目的screenshots和documentation
- 客户/利益相关者的赞扬
- 你被分配到重要项目/会议的证据
- 你的薪资历史（说明公司之前认可你的价值）
- 你的合同（特别是任何关于termination的条款）

**为什么**：一旦正式PIP开始，你的访问权限可能被收回。HR可能在某天清晨告诉你"今天起你不能访问公司系统"。

提前备份。

#### 2. 不要主动辞职

预警阶段最大的错误：

- 你感到压力大，自己提出辞职
- 公司"接受"了你的辞职
- 你失去了所有severance权利
- 你的简历显示"主动离职"

公司一定希望你这样做。这对他们最省钱。

不论压力多大，**不要主动辞职**。让公司付你离开。

#### 3. 增加你的可见性（vs 减少）

很多人在压力下变得低调，希望"别被注意到"。

这是错的。

PIP预警阶段，你应该相反：

- 主动发信息向上层汇报你的成绩
- 主动参加重要会议
- 主动建立和其他经理的关系
- 主动展示你的价值

为什么？因为如果有更多人知道你的价值，HR单独决定解雇你的成本更高。

但是注意：这种"增加可见性"必须基于真实贡献。如果你硬装，会让情况更糟。

#### 4. 开始外部机会的准备

不论你打算 fight 还是 leave，你都需要BATNA。

**立刻开始**：

- 更新简历
- 联系猎头
- 联系前同事
- 申请几份外部职位

**为什么**：

- 如果你最终通过PIP（不太可能），你也应该知道你的市场价
- 如果你没通过PIP，你需要快速找到下一份工作
- 在面试新工作时，"我目前还在职"比"我刚被解雇"议价能力强10倍

**关键时机**：在被正式告知PIP **之前**就开始找。一旦PIP开始，你的心理压力会让你的面试表现变差。

### 不要做的事

#### 1. 不要在公司大声抱怨

不要：

- 在Slack上吐槽经理
- 和同事抱怨"我被针对"
- 在公司论坛上发情绪化内容

为什么：这些都是HR收集的"unprofessional behavior"证据。即使你的判断是对的（经理确实在针对你），表达方式不专业，会被用来强化"你确实有performance issue"的论证。

#### 2. 不要直接质问经理

```
❌ "你是不是想把我搞走？"
❌ "我感觉你最近对我有意见"
❌ "你为什么不再让我参加X会议了？"
```

直接质问让经理立刻进入"防御"模式，加速他和HR的协调。

#### 3. 不要假装没事

完全相反的错误：你识别出预警，但选择"忽略"，希望"不去注意它就会消失"。

它不会消失。PIP的进程已经在进行。每一天你不行动，公司多一天准备时间。

## 第三阶段：被正式告知PIP

如果你已经被正式告知PIP（"我们要把你放进一个 Performance Improvement Plan"），策略要变。

### PIP开始时的关键决策

你有四个选择：

**选择A：尝试通过PIP**

适合：你确实可以改进 + 你判断公司是真心想让你成功 + 你想留下来。

注意：通过率约15%。即使通过，被PIP过的员工后续被裁概率仍高于平均。

**选择B：表面配合 + 私下找新工作**

适合：你判断PIP是程序性的 + 你需要时间找新工作 + 你想保留severance。

这是大多数情况下的最佳策略。

**选择C：直接谈severance**

适合：你的关系已经无法挽回 + 你想快速干净地结束 + 你能谈到合理的package。

需要勇气，但常常是最体面的选择。

**选择D：法律对抗**

适合：你有具体证据PIP是不公平的（年龄歧视、性别歧视、报复性、违反劳动法）+ 你愿意花时间和精力打官司。

罕见的选择，但在合适的情况下能拿到大额赔偿。

### 选择B的具体执行（最常见的最佳策略）

90%的情况下，你应该选B：表面配合 + 私下找新工作。

#### 表面配合的做法

1. **接受PIP，不要争辩**：

```
"谢谢您给我的反馈。我会认真对待PIP的所有目标，努力达到您的期待。"
```

不要争辩"目标不公平"、"批评不准确"。这些争辩对你的处境没有帮助，只会被记录为"defensive"。

2. **要求所有目标书面化**：

```
"为了我能高效工作，能否帮我具体定义每个目标的成功标准、衡量方式、和评估的具体时间点？"
```

这有两个好处：

- 你能清楚知道你需要做什么
- 如果目标设计不合理（不可达成的），你能在文字上看出来
- 未来如果你要法律对抗，这些文字是证据

3. **每周向经理submit进度报告**：

主动建立"我在努力"的paper trail。每周email经理：

```
本周进度：
- 目标1：[完成情况]
- 目标2：[完成情况]
- 目标3：[完成情况]

挑战：
- [遇到的困难]

需要支持：
- [你需要经理/公司提供的资源]
```

这份email有几个作用：

- 证明你"在努力"
- 把"挑战"和"需要支持"记录下来。如果公司没提供支持，这是公司"setting you up to fail"的证据
- 让经理"无法说"你没有主动沟通

4. **保存所有沟通记录**：

PIP期间所有的email、Slack、会议记录，都backup到个人邮箱。

特别注意：

- 经理的批评：是具体还是模糊
- 经理的"目标"：是具体还是模糊
- 经理的回应：是支持还是设障碍
- 同事的赞扬（即使是Slack的"好的，谢谢"）

#### 私下找新工作的做法

PIP期间是你最需要找新工作的时候。但是难度也最大（你在压力下，面试表现可能受影响）。

**找新工作的时间分配**：

- 工作日下班后：1-2小时投简历、和猎头沟通
- 周末：4-6小时面试准备
- PTO/Sick day：用来面试（不要请太多，会引起怀疑）
- 午休时间：电话面试

**面试时如何说**：

不要说"我在PIP里"。这是简历杀手。

正常的话术：

```
"我在 [公司] 已经 X 年了。我感觉到了职业的plateau，想找一个新挑战。"

"我对 [新公司/新职位] 感兴趣，因为 [具体原因]。"

"目前我的工作还是稳定的（进行时态），但我在主动寻找下一步。"
```

注意：避免直接撒谎（"我没问题"）但也不主动暴露弱势（"我在PIP里"）。模糊处理。

**关键时间点**：

如果你能在PIP的60天内拿到外部offer，立刻接受新offer，向经理提辞职。

这样你：
- 避免了"被解雇"的简历污点
- 可以谈到对方的入职日期晚一些（让你做完目前的PIP）
- 离开时是"找到新机会"，而不是"被裁"

如果60天没拿到offer，进入下一个阶段。

### 选择C：直接谈Severance

有时候，最好的选择是直接和HR谈：

"我们都知道PIP的结局。让我们谈一个体面的离开方案。"

这需要勇气。但常常对双方都好：

- 公司不需要执行PIP的60-90天
- 你不需要承受PIP的精神压力
- 你能拿到severance package
- 你可以专注找新工作

#### 何时应该选C

- PIP的目标明显不可达成
- 经理的态度让你确信他想让你走
- 你已经心理崩溃，无法集中精力工作
- 你判断你不会通过PIP

#### 谈Severance的话术

不要在第一次PIP会议后立刻谈。给自己几天思考。

然后约一个会议（最好和HR + 经理）：

```
"谢谢您们给我PIP。在仔细思考后，我想坦诚地讨论一下。

我感受到了团队的方向已经发生变化，我可能不再是这个职位的最佳人选。我尊重这个判断。

但我希望我们能找到一个对双方都体面的方式。我提议：

我们停止PIP流程。我同意在 [日期] 离开公司。作为交换，我希望讨论一个severance package。

具体来说：
- X个月的基本薪资作为severance（标准是1-3个月，但如果你tenure长，可以争取6-12个月）
- 我未休的假期按现金兑换
- COBRA健康保险延续3-6个月（美国） / 类似福利
- 我的股票vesting加速 [如果适用]
- 一份正面的离职证明，让我能在新工作面试时使用
- 一份neutral reference policy（公司不主动说我的坏话）

作为交换，我会签一份standard separation agreement，不会起诉公司，不会公开谈论这次离开。

我希望我们能在 [一周] 内达成agreement。"
```

#### 谈判的关键点

1. **永远在会议之前email你的提议**：把要求写清楚，让HR有时间准备。
2. **使用"对双方都好"的框架**：让HR觉得这是公司的胜利，不是损失。
3. **准备好你的BATNA**：如果谈不好severance，你的下一步是什么？（继续PIP？起诉？）
4. **不要立刻接受HR的第一次回应**：一定有谈判空间。
5. **要求把所有承诺写入Separation Agreement**：口头不算。

### 选择D：法律对抗

只在以下情况考虑：

- PIP明显是基于年龄、性别、种族、宗教的歧视
- PIP是对你"提出合法投诉"的报复（whistleblower retaliation）
- PIP违反具体劳动法（你所在国家/地区）
- 你有书面证据支持上述指控

如果决定法律对抗：

1. 立刻找专业的劳动律师（不是普通律师）
2. 不要自己直接和HR对抗（让律师代理）
3. 记录每一次和公司的互动
4. 保留所有书面证据
5. 准备一年的精神和经济成本

法律对抗成功率因情况差异巨大。在合适的case下，可以拿到几十万到几百万美元的赔偿。但代价也大。

## 第四阶段：PIP之后的恢复

不论PIP的结果如何，你都需要从这个经历中恢复。

### 如果你"通过"了PIP

恭喜，但保持警觉。

数据显示：通过PIP的员工，未来12个月内被裁的概率仍然是平均的2-3倍。

公司会用更"温和"的方式让你离开（重组、岗位调整、第二次PIP）。

**建议**：把通过PIP当作"buy time"。继续找新工作。在你的下一个window期主动离开。

### 如果你被解雇了

**第一周：处理实际问题**

- 申请失业保险（如果适用）
- 处理健康保险（COBRA / 个人保险）
- 评估你的financial runway
- 更新简历，开始申请

**第一个月：心理恢复**

被裁不是道德失败。你不是bad person。

但情绪上你会经历：

- 愤怒（"这不公平！"）
- 否认（"也许我做错了什么..."）
- 自责（"我应该早点行动"）
- 抑郁（"我永远找不到好工作了"）
- 接受

不要在这些情绪里停留太久。每一周都是机会成本。

如果情绪压垮你，找心理咨询。这是合理的需要，不是软弱。

**找新工作时的话术**

面试官会问"为什么离开上一家？"

不要说："我被裁了。"

但是更不要撒谎（公司可以查reference）。

中性话术：

```
"我和 [公司] 在 [日期] 分开。原因是 [组织重组/团队方向变化/我和经理在工作风格上的差异]。我现在在寻找下一个挑战。"
```

注意：

- 不主动说"被裁"，但不撒谎
- 用结构性原因（重组、团队变化）而不是个人原因（我表现不好）
- 不抱怨前公司
- 快速转向"我希望做什么"

### 长期恢复

PIP和被裁会留下心理痕迹。可能持续1-2年。

健康的处理方式：

- 不要把PIP内化成"我是loser"
- 理解PIP的统计本质（85% 的人最终离开 = 这是流程，不是评价）
- 在新工作里，不要因为PIP创伤过度紧张
- 但是保留"对管理者动机的清醒判断"。这是PIP给你的最重要礼物

很多人在PIP后变得更clearheaded。他们再也不会盲目相信"我们是一家人"的话术。这种清醒，是巨大的资产。

## 关键建议

### 给所有读者的提醒

1. **不要等到PIP才警觉**：识别预警信号，提前行动
2. **保留所有证据**：从你入职第一天起
3. **建立外部网络**：不要把所有职业能量投在一家公司
4. **永远保持BATNA**：每年至少一次外部面试
5. **理解你和公司的关系**：是商业交易，不是道德关系

### 给PIP预警中的读者

立刻做这5件事：

1. 备份所有正面证据到个人邮箱
2. 更新简历
3. 联系猎头和前同事
4. 不要主动辞职
5. 增加你的可见性（基于真实贡献）

### 给已经在PIP里的读者

90%的情况，选择B：表面配合 + 私下找新工作。

记住：通过率15%。理性决策，不要被情绪驱动。

如果你判断PIP不可通过，认真考虑选择C：直接谈severance。这往往是最体面的方式。

不要主动辞职（除非已经拿到新offer）。让公司付你离开。

### 给已经被解雇的读者

你不是失败者。你是一个被错位匹配 / 被结构性变化波及的人。

- 处理短期实际问题
- 给自己时间心理恢复
- 用学到的"清醒"装备下一段职业
- 这次经历会让你更强

## 最后的话

PIP是资本最阴险的工具之一。

它把"解雇你"包装成"帮助你"。它把"程序化的暴力"叫做"关怀"。它要求你在自己被处刑的过程中"配合改进"。

理解这一点不是为了让你愤怒。是为了让你清醒地保护自己。

世界上最重要的人不是公司HR。是你自己。

世界上最重要的资产不是你的工资。是你的健康、你的尊严、你的下一份工作的机会。

PIP的所有压力，都是公司转嫁给你的。它不应该成为你的全部。

走出PIP，无论以什么方式，都意味着你有了一次"看清楚资本-劳动关系真实结构"的机会。

这种清醒，是它给你的唯一礼物。

请收下这份礼物，并把它传给你的同事、你的朋友、你的下一代。

让他们知道：当公司说"我们关心你的成长"时，他们应该问："具体怎么关心？写下来。"
