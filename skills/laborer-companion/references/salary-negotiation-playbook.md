# 薪资谈判脚本 / Salary Negotiation Playbook

市场上99%的薪资谈判指南是写给职场赢家的：如何拿到更多。

这份文档不是。

这份文档是写给被压榨的劳动者的：如何不被压低、如何识破公司的谈判陷阱、如何拿到你应得的份额。

---

## 谈判前必须知道的真相

### 真相一：公司HR是受过训练的职业谈判者

你今年也许只谈一次薪资。HR每周都在谈。她谈过几百次。她有完整的话术库、心理学训练、行业数据。

你和HR的谈判是不对称的。除非你也做了准备。

### 真相二：公司给你的"区间"通常是假的

当HR说"这个岗位的预算是 X 到 Y"，真相是：

- X 和 Y 都是HR自己决定的
- 实际上公司可以给得更高（如果你足够强势）
- "区间"的目的是把你的期望锚定在区间内，而不是告诉你真实预算

公司真正的预算上限，通常比HR给你的"区间上限"高 20-40%。

### 真相三：先开口的人通常输

在薪资谈判心理学里有一个铁律：第一个说出具体数字的人，定下了讨论的"锚"。

如果你先说"我希望年薪 15 万"，公司大概率会让你接受 13-15 万。
如果HR先说"我们能给 13 万"，你大概率会争取 13-15 万。

注意两种结果都把你压在 13-15 万。锚定的位置决定了讨论的范围。

### 真相四：你的议价能力，在拿到offer前最高，签完后最低

谈判最佳窗口：

```
拿到offer ━━━━━━━━━━━━ 接受offer ━━━━━━━━━━━━ 入职 ━━━━━━━━━━━━ 第一次review
   ↑                          ↓                       ↓                        ↓
最高议价权              快速下降                 较低                     重新升起（如有外部offer）
```

很多人在拿到offer的瞬间太激动，立刻接受。这是把自己最强的牌打掉。

正确的做法：拿到offer后，**至少48小时**才回应。

### 真相五：跳槽是大多数人涨薪的最快方式

数据：内部加薪平均每年3-5%，跳槽涨幅平均15-30%。

这不是公司"奖励忠诚"的失败。这是结构性的：

- 你在公司内部，HR用"内部公平"来限制你的涨幅
- 你跳槽时，新公司用"市场价"来定你的工资
- 这两个数字差距巨大

如果你已经在一家公司5年没涨过薪，你的市场价可能比你现在的工资高30-50%。

## 子模块 7A：跳槽时的薪资谈判

### 第一阶段：准备（offer前）

#### 1. 调研真实市场价

不要只看Glassdoor的平均数。要看：

- **levels.fyi**（科技行业最权威，有具体公司具体级别的真实数据）
- **Blind**（匿名社区，工程师真实薪资分享）
- **本地论坛**（小红书、脉脉、teamblind等）
- **直接问**（你认识的、在目标公司或类似公司工作的人）

调研出三个数字：

```
P25（25%分位）：你这个level在这家公司能拿的下限
P50（中位数）：标准水平
P75（75%分位）：上限。你的目标
P90（90%分位）：极限。你和强offer competing时可能达到
```

你的目标是 P75。不是 P50。

#### 2. 计算你的BATNA（Best Alternative to Negotiated Agreement）

BATNA是你"如果谈不成，下一个最好的选项"。

如果你目前的工作还可以，BATNA = 留下来。
如果你已经决定离开目前的工作，BATNA = 等下一个offer。
如果你有多个offer，BATNA = 第二好的offer。

**BATNA越强，你的议价能力越强。**

如果你的BATNA很弱（必须立刻找到工作），不要让对方知道。

#### 3. 不要先问薪资范围

很多人在第一轮电话面试就问"这个岗位的薪资范围是多少"。

这是一个可以理解但战略上次优的做法。原因：

- 你过早暴露你对薪资敏感
- 你接受了对方的"区间"作为讨论框架
- 你失去了"让对方先报价"的机会

**正确做法**：在面试过程中专注展示你的价值。让对方对你的兴趣升高。等到他们要给offer时，让他们先报价。

如果对方非要你先报价：

❌ 错误："我希望年薪15万"
✅ 较好："我现在的总包是X，我希望这次跳槽能有合理的提升"
✅ 最好："我对这个职位本身很感兴趣，我相信贵司会基于这个职位的市场价值和我能带来的贡献给出有竞争力的offer"

第三种回应把球踢回去。让HR先报价。

### 第二阶段：拿到offer时

#### 1. 不要立刻回应

不论offer是好是坏，第一反应应该是：

"非常感谢这个offer。我对加入贵司很期待。能否给我X天时间仔细评估？我希望做出对双方都有意义的决定。"

X = 48小时（最短）到 7 天（最长）。

如果HR施压"我们今天就需要答复"，参考前面rhetoric-patterns里的"紧迫感操控"应对方式。

这48小时到7天里，你要做几件事。

#### 2. 拆解offer的真实价值

公司给的offer里有很多包装。要把它拆开看：

```
真实年总包 = 
  基本薪资 +
  保证奖金（不是"target bonus"，是"保证拿到的部分"）+
  签字费 / 4 (如果分4年vesting) +
  RSU（按当前股价）/ vesting年数 +
  401k匹配 +
  其他确定的福利（搬家费、HSA匹配等）

不要算进去：
- target bonus（目标奖金，公司随时可以以performance为由不给）
- 期权/options（如果不是已经IPO的公司，期权的真实价值通常接近零）
- "无限假期"、"成长机会"、"灵活工时"等无法量化的"福利"
```

这个真实数字，是你比较offer的基础。

#### 3. 识别offer里的陷阱

**陷阱A：股票vesting的"4年4年"**

很多公司给"4年vesting，第一年cliff 25%"。这看起来标准。但要注意：

- 如果你4年内离开，未vested的股票全部归公司
- 这是一种"长期把你绑定"的设计
- 如果4年里公司业绩下滑、股价下跌，你的股票可能价值远低于预期

**应对**：把股票按"假设到手时打50%折扣"来计算。

**陷阱B：签字费的"如果1年内离开要全额退还"**

签字费（sign-on bonus）通常有"留任条款"。如果1年（有时2年）内你离开，要按比例退还。

**应对**：把签字费按"必须留X年才能拿"来评估。如果你不确定能在公司待这么久，签字费的真实价值要打折。

**陷阱C：Total Comp 表面光鲜的"包装数字"**

"Total compensation: $300K"听起来很好。但拆开可能是：

- 基本薪资：$150K
- Target bonus 20%（不保证）：$30K
- RSU 4年vesting，今年价值 $80K
- "Annual refresh" RSU 假设 $40K（这是HR编出来的）

实际你今年能拿到的是：$150K + 一些不确定的奖金 + $20K（第一年cliff之前你拿不到）= ~$170-180K

不是 $300K。

**应对**：要求HR给你"年化的、按现金流计算的"年总包预测，分年列出。

#### 4. 决定还价策略

收到offer后，分析：

- 这个offer相对市场价（你之前调研的P50/P75）如何？
- 你有没有其他offer做BATNA？
- 你有多想要这家公司？

基于这个分析，决定还价的目标。

**还价的两条铁律**：

铁律1：永远要还价。 即使offer已经超出你的期望，也要还价。原因：
- 公司给你的offer通常有10-20%的"可让空间"
- 不还价的人在HR眼里是"软柿子"，未来加薪也容易被压低
- 还价不会让公司收回offer（除非你提了不合理的数字）

铁律2：还价要有依据，不要哀求。 不说"我希望多一点"。说"基于X、Y、Z我希望调整到N"。

#### 5. 还价的具体脚本

**场景：你拿到一个offer，你想还价**

```
"非常感谢这个offer。在仔细考虑之后，我有一些反馈想和您讨论。

基于我的研究和我了解的市场情况，类似经验的[你的级别]在贵司的薪资范围中位数是[你查到的P50]。考虑到我[列出你的具体优势：经验年限、特定技能、过往成绩]，我希望我们能讨论一个反映这些价值的offer。

具体来说，我希望：
- 基本薪资从 X 调整到 Y（数字基于你的调研，比offer高10-20%）
- 签字费从 A 调整到 B（如果有）
- 股票从 M shares 调整到 N shares（如果有）

这不是关于'我想要更多'，而是关于让offer的整体价值反映这个职位的市场水平和我能带来的贡献。

我对加入贵司非常期待，也希望我们能找到对双方都合适的方案。"
```

**关键技巧**：

1. 一开始感谢，但不要过度感谢（不要让对方觉得你已经满意了）
2. 用具体数字和市场数据（不是个人感受）
3. 具体说每一项的目标数字（让谈判可执行）
4. 强调"对双方都合适"（暗示如果不合适你不会接受）
5. 表达"期待"但不"必须"（保留BATNA的暗示）

#### 6. 应对常见的回应

**回应A："这是我们能给的最高了"**

90% 概率是假的。回应：

```
"我理解可能有内部预算约束。但是基于市场数据，这个offer低于这个职位的中位数。我希望我们能找到一个解决方案。如果基本薪资暂时无法调整，能否在签字费/股票/其他维度上调整？"
```

把谈判从"基本薪资"扩展到多个维度。公司在某个维度上的灵活性通常比另一个维度大。

**回应B："如果你不接受，我们就给下一个候选人了"**

这是紧迫感操控。回应：

```
"我尊重您的时间。我也希望快速做出决定。但这是一个重要决定，我希望基于充分的信息考虑。如果48小时内的回应对您不可行，我尊重您的选择。"
```

90% 的情况，公司会等。如果他们真的不等，那这家公司从一开始就不是一个好选择。

**回应C："你的薪资期望超出了这个职位的预算"**

回应：

```
"我理解。我的期望基于市场数据[再次提供你的依据]。如果这个数字超出预算，能否帮我理解：贵司认为我比市场中位数低 X% 的原因是什么？这能帮我评估是否有调整我自己期望的依据。"
```

把"是否合理"的责任推回给对方。让他们解释为什么offer低于市场。

### 第三阶段：终局

#### 1. 知道何时停止

谈判通常有2-3轮。再多就过了。

如果对方第二轮回应已经接近你的目标，接受。如果完全不动，你要决定：

- 接受现状
- 用其他offer做杠杆（如果有）
- 拒绝offer（如果你有BATNA）

不要无限制地谈下去。HR会开始觉得你"难搞"，未来在公司内会被记住。

#### 2. 让所有承诺书面化

任何HR口头承诺的东西，要求写在offer letter里：

- 签字费具体数字和支付时间
- 股票数量和vesting schedule
- 薪资具体数字
- 起始日
- 任何"我们会在X月给你Y"的口头承诺
- 远程工作政策
- 任何承诺的福利

**HR说的不算。文字写下来的算。**

如果HR拒绝把某条承诺写进offer letter，那条承诺不存在。

#### 3. 接受offer的话术

```
"非常感谢您和贵司在这个过程中的努力。我接受这个offer。

我希望确认一下最终条款：[列出每一项具体数字]。

期待我们的合作。"
```

短，专业，不再讨价还价。

## 子模块 7B：在职加薪谈判

在职加薪比跳槽涨薪难。原因：

1. 公司没有"市场竞争"压力（你已经在他们手里）
2. HR有"内部公平"的限制（不能给你比同级别同事高太多）
3. 你的BATNA较弱（如果不接受，难道辞职？）

但是仍然有空间。

### 第一阶段：选择时机

**最佳时机**：

1. **完成一个重大项目之后**：你的价值刚被证明
2. **公司财报好之后**：公司有钱
3. **行业招聘旺季**：你的市场价高
4. **你拿到外部offer之后**（最强）

**最差时机**：

1. 公司刚裁员后
2. 经济衰退期
3. 你刚搞砸一个项目后
4. 你的经理刚换人（新经理需要时间评估你）

如果你目前不在最佳时机，等。不要在最差时机谈。

### 第二阶段：准备数据

加薪谈判最重要的不是"你想要多少"，是"你有什么数据支持"。

需要的数据：

#### 1. 你过去一年的具体贡献

不是抽象的"我做得不错"。是具体的：

- 我负责了X项目，带来Y收入/节省Z成本
- 我解决了W问题，避免了V损失
- 我培训了U新人，提升了团队效率
- 我做了T创新，被公司P部门采用

把这些列成3-5条。每条都有数字支撑。

#### 2. 市场数据

你这个级别在外部市场的薪资是多少？（用levels.fyi、Blind等）

如果你的工资低于市场，这是最有力的数据。

#### 3. 内部公平数据

如果可能，知道公司内部同级别同事的薪资。（通过Blind、私下交流、leaked data）

如果你的工资低于内部同级别，这也是有力的数据。

#### 4. 你的scope扩展

过去一年里，你的实际工作内容是否扩大了？

- 接手了原本是另一级别的工作
- 承担了离职同事的责任
- 处理了scope外的紧急问题
- 在跨team合作中起了主导作用

这些是你"实际级别"高于"名义级别"的证据。

### 第三阶段：开口

#### 1. 不要在年度review时谈

年度review的薪资调整是HR预先决定的（通常占公司总工资的3-5%作为加薪池）。在这个时点谈，HR的回答永远是"我们已经按policy处理了"。

**最佳时点**：在review前的几个月（比如review在12月，你应该在8-10月谈）。这给经理时间向上申请exception budget。

#### 2. 用email提出要约

不要在1:1时突然提。这让经理措手不及，他第一反应通常是防御性的。

发一封email，给经理时间准备：

```
主题：希望讨论我的薪资和级别

[经理姓名]：

希望和您讨论一下我目前的薪资和级别情况。

过去一年里，我[列出3-5条具体贡献，每条带数字]。

基于这些贡献和我了解的市场数据，我认为我目前的薪资有调整空间。具体来说，我希望讨论：

- 基本薪资从 X 调整到 Y（涨幅约 N%）
- 或者 提级到 [下一级]，配套相应的薪资调整

我希望我们能找一个时间深入讨论。我已经准备了具体的数据和理由，会带到会议里。

希望听到您的想法。

[你的名字]
```

为什么用email：

1. 让经理有时间准备（不用即兴回应）
2. 让经理可以转给HR作为申请材料
3. 给你一个"白纸黑字"的paper trail
4. 避免1:1时的情绪化回应

#### 3. 谈话时的关键技巧

**A. 始终用"价值贡献"框架，不是"我需要"框架**

❌ 错误："我房贷压力大，需要加薪"
✅ 正确："我对公司的贡献已经超过了目前的薪资定位"

公司不在乎你的家庭情况。公司只在乎你的价值。

**B. 让经理成为你的盟友，不是对手**

经理夹在你和HR之间。如果经理觉得你给他制造麻烦，他不会帮你争取。

```
"我理解经理您也在制度内工作。我希望我们能一起想办法，向上层证明这个调整的合理性。我能提供哪些数据/材料帮助您做这个case？"
```

把"我vs公司"变成"我们vs HR/上层"。

**C. 准备好回应"现在不是时候"**

经理常用回应：

- "公司今年财务紧张"
- "现在不是合适的时机"
- "下个quarter我们再看"
- "和HR的budget cycle不一致"

应对：

```
"我理解时机的考虑。能否具体说：什么时候是合适的时机？需要什么条件出现？我希望我们能定一个具体的时间点和评估标准，这样我能高效准备。"
```

让"暂缓"具体化。如果经理拒绝具体化，"暂缓"就是"永远不"。

**D. 准备好打BATNA牌**

如果你有外部offer，最强的牌：

```
"我应该坦诚一下。我最近在外部市场也看了一些机会，收到了一些信号显示我的市场价比目前的薪资高。我的首选是留在贵司，但我也需要在职业上做出对自己最好的决定。我希望我们能找到让我留下来的方案。"
```

**注意**：只有当你真的有外部offer时才能这么说。如果是bluff，被识破后果严重。

### 第四阶段：被拒绝后的应对

不是所有加薪请求都成功。被拒绝后的应对决定你的下一步。

#### 1. 让"拒绝"具体化

```
"我理解这次没法调整。能否帮我具体说明：
1. 拒绝的具体原因是什么？（是budget、是performance、是公司policy？）
2. 如果我做X、Y、Z，下次能调整吗？请给具体的标准。
3. 下次评估的时间点是什么？"
```

如果经理给不出具体答复，你已经知道答案：**这家公司不会给你加薪**。开始找外部机会。

#### 2. 如果是"performance"原因

要求：

- 具体的performance gap是什么
- 多长时间内改善能达到加薪标准
- 有没有人帮助你（mentor、培训、资源）

如果对方拒绝具体化，这是借口不是理由。

#### 3. 如果是"budget"原因

问：

- 公司的整体加薪池是多少？我的份额是怎么决定的？
- 同级别同事得到了多少？
- 下个financial year budget会改善吗？

如果你坚持问，HR可能会透露真相：你的"budget"问题其实是"你不在optimal hire list上"。

#### 4. 决策点

被拒绝后，你有三个选项：

**选项A：接受并继续工作**
适合：你的BATNA弱、约束大、还需要这份工作的稳定。
但是：你要警惕，未来再被拒绝几次后，你的"市场价"会被压低。

**选项B：开始找外部机会**
适合：你的BATNA可以变强、你的技能在市场上有价值。
最有效的"加薪"通常是跳槽。

**选项C：硬碰硬**
适合：你已经决定离开、想拿一份漂亮的离职说法。
做法：提交辞职 + 等公司挽留 + 谈一个高于目前的薪资。
风险：参考"counter-offer"那一节，70%的人在18个月内被裁。

我个人推荐 **B**。这是最理性的策略。

## 子模块 7C：识别压榨型offer

不是所有offer都值得接受。有些offer表面上看不出问题，但接受了就是上当。

### 红旗信号清单

**红旗 1：薪资明显低于市场**

如果offer给的工资低于levels.fyi或类似数据的P50，这家公司认为他们能用低于市场的价格雇到你。

可能性：
- 公司认为你议价能力弱
- 公司财务紧张但隐瞒
- 公司以"低价hire大量人"为策略

不论哪一种，对你都不是好兆头。

**红旗 2：用"成长机会"换低工资**

"我们工资不高，但是这是一个超棒的成长机会"。

这句话翻译成实话："我们想用低于市场的工资雇你，因为我们觉得你年轻没经验"。

成长机会和合理工资不是对立的。一家真正能给你成长的公司，能给你成长 + 合理工资。

**红旗 3："灵活工时"代替明确合同**

"我们没有固定上下班时间，你想几点工作都行。"

听起来很灵活。实际意思："我们期待你随时在线，没有加班费的概念"。

**红旗 4："无限假期"政策**

"无限假期"在数据上的真实结果是：员工平均休假天数比"标准假期"政策的公司更少。

为什么？因为没有明确额度，员工心理上会自我审查"休多少合适"。结果休得比应得的少。

公司用"无限"包装"实际更少"。

**红旗 5：股票期权占offer大头但公司不上市**

如果一家未上市公司的offer里：

- 基本薪资低于市场
- "总包"看起来高，但80%来自stock options/RSU

那这个offer的真实价值取决于公司是否上市/被收购。

历史数据：未上市startup最终给员工带来"足够生活"的股票回报的概率，低于10%。

如果你接受这种offer，你在押公司的成功。这是一笔投机性投资，不是工资。

**红旗 6：合同里有过度的非竞业条款**

非竞业（non-compete）条款限制你离职后不能去竞争对手公司。

如果offer里的非竞业条款是：

- 范围太广（比如"任何相关行业"）
- 时间太长（比如"24个月"）
- 没有补偿（你不能工作但公司不付钱）

这家公司想锁定你，不是雇佣你。

注意：在加州，非竞业条款基本不可执行。在中国，有些非竞业条款不付补偿是无效的。但合同里写了，至少在心理上压你。

**红旗 7：HR的态度变化**

面试时HR对你殷勤热情。Offer给了之后突然冷淡，开始用"程序"压你。

这是一个微妙的信号。它说明HR觉得你已经"上钩"，开始切换到"管理"模式。

如果未来你在公司有任何议价需求，会面对的是这个冷淡的HR。

**红旗 8：经理过度强调"culture fit"**

如果经理在面试里反复强调"我们公司的culture是X"，特别是X包含"all hands on deck"、"family"、"hustle"等词，这是一个warning。

这种公司常常用culture作为话术，让你接受合同范围外的工作。

**红旗 9：复杂的奖金结构**

"基本薪资 X + target bonus 30%"。这个30%听起来很多。但是：

- "target"不是"保证"
- 实际bonus基于"performance"、"公司业绩"、"team performance"等多个不在你控制下的因素
- 历史上发达国家的target bonus实际兑现率约 50-70%

所以这个 30% 真实价值约 15-21%。HR用"target"包装，让offer看起来比实际高。

**红旗 10：拒绝把口头承诺写入offer letter**

面试时HR说："你前6个月会有特别的onboarding，确保你成功"、"我们会在3个月内给你具体的晋升路径"。

但是offer letter里没有这些。

如果你要求把这些写进letter，HR说"这些是惯例，不需要写进合同"。

90%概率，这些口头承诺不会兑现。如果不会兑现，为什么不能写进合同？

### 应对策略

#### 1. 红旗1-3个：可以谈判

如果你识别出1-3个红旗，可以尝试在谈判里解决：

- 红旗1（低工资）→ 还价到市场价
- 红旗3（灵活工时）→ 要求明确的工作时间和加班政策
- 红旗5（股票为主）→ 要求更多基本薪资，少股票

如果公司愿意调整，可以接受。如果坚决不调整，进入下一步。

#### 2. 红旗4-6个：严肃考虑拒绝

红旗多说明公司的整个雇佣框架有问题。即使你"谈判成功"修复了几个，其他的还在。

考虑：

- 你有其他offer吗？
- 你的BATNA是什么？
- 一年后这家公司还会让你后悔接受吗？

如果你的BATNA强，拒绝。

#### 3. 红旗7个以上：直接拒绝

当一个offer有7个以上红旗，这不是一个普通的"压榨型offer"。这是一家系统性剥削员工的公司。

接受这种offer的代价：

- 入职后立刻进入被剥削状态
- 公司会用"试用期"等理由进一步压你
- 一年后你的简历会显示"刚做一年又跳槽"，对未来面试不利
- 你的职业资本会被这家公司消耗

直接拒绝。

## 子模块 7D：补偿性跳槽（被低薪雇佣多年后）

这是一个特殊场景。

如果你在一家公司被低薪雇佣很多年，跳槽时面对的特殊挑战是：

1. 你的"目前薪资"很低，新公司想用"目前薪资 + 20%"的逻辑给你offer
2. 这意味着你过去几年被压低的工资，会"持续"到下一份工作
3. 你需要打破这个循环

### 策略：把"应得薪资"作为锚

新公司面试时，HR必然问："你目前薪资是多少？你的薪资期望是多少？"

如果你直接回答目前薪资，你就让对方拿了"低锚"。

**正确回应**：

```
HR："请问你目前的薪资是多少？"

你："我希望讨论这个职位的薪资范围，而不是我目前的具体数字。我目前的薪资反映的是历史决定，不一定反映这个职位的市场价值。我希望我们基于这个职位本身、市场数据、和我能带来的贡献来讨论。

如果您一定需要参考数据，我可以提供：基于我的级别和行业数据，市场范围是 X-Y。"
```

把讨论从"你过去拿多少"转移到"这个职位值多少"。

### 应对"我们的policy是参考你目前薪资"

如果HR坚持要知道你目前的工资，并以此为基准：

```
"我理解贵司有内部policy。但我希望诚实地说一下我的考虑：

我目前的薪资是 X，这是我入职5年时的薪资，期间只增加过 Y%。市场已经发生了显著变化，我的能力也在 5 年里提升了。

如果 offer 完全基于我目前的薪资，本质上是延续了一个已经过时的定价。这对双方都不是最优的：

- 对您来说，您拿到的是一个被低估的员工，他知道自己被低估，会持续找新机会
- 对我来说，我无法在贵司开始一个真正符合我市场价值的雇佣关系

我希望我们能基于这个职位的市场价、我的能力，和我能贡献的价值，重新定价。"
```

这是一个真诚的、对双方都好的论证。优秀的HR会理解。

### 备选：拒绝透露目前薪资

在很多地区（加州、纽约、马萨诸塞、加拿大部分省份），雇主询问求职者目前薪资是非法的。

即使在不违法的地区，你可以礼貌地拒绝：

```
"我倾向于不分享我目前的具体数字。我希望我们的讨论基于这个职位的市场价值。"
```

如果HR坚持要数字，他可能不是一个值得加入的雇主。

### 谈判时的"时间价值"论点

如果你被低薪了3-5年，你不只是损失了那几年的钱。你损失了：

- **复利效应**：低工资带来的低储蓄/低投资带来的低长期财富
- **职业发展**：低工资公司通常成长机会也少
- **市场信号**：未来雇主看到你的低工资，会按低工资定价

跳槽时，新公司的offer应该补偿这些损失，而不是延续它。

**具体话术**：

```
"我希望诚实地说，过去5年我可能没有按市场价被定价。这部分历史已经无法改变。我希望接下来的5年能在公平的市场定价下工作。

这意味着我希望的不是'目前工资 + 20%'，而是'这个职位的P75'。我相信这个数字是 X-Y。希望我们能在这个范围内讨论。"
```

## 关键提醒

### 关于"贪婪"的迷思

很多人在谈判时心理上的最大障碍是："我会不会显得太贪婪？"

**真相**：

1. 你和公司的雇佣关系是商业交易，不是道德关系
2. 公司在和你谈判时不会担心"显得贪婪"。他们的HR目标就是用最低的工资雇用你
3. 你不为自己谈判，没有人会为你谈判

不要把"为自己争取应得的"包装成"贪婪"。这是资本最希望你内化的话术。

### 关于"对公司不忠诚"的迷思

跳槽 = 不忠诚？这是一个被资本反复灌输的迷思。

**真相**：

1. 你的雇主每天都在评估"是否应该裁掉你"。这没有"忠诚"问题
2. 公司经历多次裁员、合并、重组、出售，从来不会因为"老员工的忠诚"而保留谁
3. 你对公司的忠诚，公司不会对你忠诚来回报

理性的看法：雇佣是双方的市场交易。你和公司每天都在评估这个交易是否值得继续。这不是道德问题，是经济决定。

### 关于"够用就行"的迷思

很多人觉得："我现在的工资够用了，何必为了多20%去谈判"。

**真相**：

1. 工资差距的复利效应巨大。今天少20%，10年后可能是几十万的差距
2. 你不为自己谈判，HR记住的是"这个人可以被压低"。下次加薪/晋升你也会被压低
3. 谈判技能本身是一项重要的人生技能。每谈一次都是练习

不要让"够用"成为你不为自己争取的借口。

---

## 总结

谈判不是关于贪婪。是关于让你的工资反映你的真实价值。

每一次成功的谈判，都是对那些没有谈判能力的同事的一次帮助：

- 你提高了你这个级别的"市场价"
- 你向HR证明了"这种员工不能被压低"
- 你给身边的人树立了榜样

谈判是个人行为，但有集体意义。

那些不为自己谈判的人，等于在帮资本压低整个劳动者群体的工资。

你为自己谈判，就是在为你的同事谈判，为你的下一代谈判。

不要把这件事看小了。
