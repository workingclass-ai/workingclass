# 找工作和面试剧本 / Job Search & Interview Playbook

找工作是劳动者一生中最不对称的权力时刻之一。

公司有面试官培训、招聘心理学顾问、规模化面试经验、强大的背景调查工具。
你只有你自己，加上你能在网上找到的散乱建议。

这份文档的目的，是把劳动者在求职过程中容易踩的所有坑系统化，让你看见公司的招聘机器是怎么运作的，并知道如何在这个机器里保护自己的议价权。

---

## 核心原则

求职过程中，你不是在"卖自己"。

你是在和未来的雇主进行一场商业谈判。这场谈判的结果，决定了你接下来3-5年的工资、健康、家庭时间、职业方向。

如果你用"卖自己"的姿态，你会被压低。
如果你用"商业谈判"的姿态，你才有机会拿到合理的条件。

下面所有的具体建议，都是从"商业谈判"的角度展开的。

---

## 子模块 9A：找工作的战略决策

在投第一份简历之前，先想清楚几个问题。

### 你是被动找还是主动找？

**被动找**（你被裁、被PIP、内部冲突让你不得不走）：

- 时间紧迫
- 议价权弱（雇主可能闻到味道）
- 心理状态可能不佳
- 优先级是"快速找到下一份"而不是"找到最好的"

**主动找**（你目前工作稳定，但想成长/加薪/换方向）：

- 时间充裕
- 议价权强（你不急）
- 心理状态稳定
- 优先级是"找到对的那个"，不是"快"

这两种situation的策略**完全不同**。下面分别讲。

### 被动找的策略

**第一周做的事：**

1. **保护好现状**
   - 不要立刻辞职。让公司付钱让你离开（参考PIP模块的severance谈判）
   - 不要在LinkedIn上立刻把status改成"open to work"。你现在的雇主可能在监控
   - 不要在工作设备上做任何求职相关的事

2. **盘点资源**
   - 你有多少存款？能撑几个月？
   - 你的健康保险什么时候失效？
   - 你的签证状态（如果适用）会怎样？
   - 你最近一次更新简历是什么时候？

3. **重启网络**
   - 联系前同事（用私人channel，不是LinkedIn message）
   - 联系猎头（特别是那些以前找过你的）
   - 联系大学校友、行业朋友
   - 大多数好工作不是通过申请找到的，是通过网络

4. **更新简历和LinkedIn**
   - 简历更新但LinkedIn暂时不动
   - 简历用Word格式，便于ATS（应聘者跟踪系统）解析
   - 在简历里突出可量化的成就（不是job description）

**前两个月做的事：**

5. **批量申请 vs 精准申请**
   - 不要陷入"批量申请"的陷阱（每天投50份很爽但效率低）
   - 每周精准申请10-15个真正合适的职位
   - 每个职位前花1-2小时定制简历和cover letter

6. **平行处理**
   - 不要等一家公司有结果再申请下一家
   - 同时和5-10家公司在不同阶段
   - 这给你BATNA和议价权

7. **每天保持节奏**
   - 早上：申请新职位（1-2小时）
   - 下午：回应面试邀请、做面试准备
   - 晚上：休息、保护身心

### 主动找的策略

**主动找的核心智慧是：不急**。

你的BATNA是"留下来"。这意味着：

1. **目标提高**：不要随便接受一个"略好"的offer。要找一个明显更好的
2. **时间放长**：3-6个月的找工作周期是合理的。不要给自己人为的截止日期
3. **谈判更狠**：每一次都把它当作"如果谈不成我就留下"。这种心态让你能拿到更好的条件
4. **保密更重要**：你目前的雇主不应该知道你在看。一旦泄露，你的内部position会变差

**主动找的禁忌：**

- 不要因为"被市场情绪带动"就跳。市场好的时候大家都在跳，你跳的目的应该清晰
- 不要因为"老板让我不高兴一次"就跳。老板每天都让人不高兴
- 不要因为"同事跳了我也要跳"就跳

跳槽是10年职业的事，不是这个月的情绪。

## 子模块 9B：识别"伪机会"和招聘陷阱

市场上大量职位是**"幽灵职位"**或者**"陷阱职位"**。

识别它们能让你节省巨大时间，避免被剥削。

### 幽灵职位：从来没打算招的职位

**信号 1：职位挂了3个月以上还没招到**

公司可能：
- 在"摸市场"看有什么人才（但实际不打算招）
- 在收集简历做未来的招聘库
- 老板想招但budget没批
- 内部已经有候选人，挂职位是为了"程序合规"

如果你看到一个挂了很久的职位，**直接问HR**："这个职位还在active recruiting吗？大概什么时候会做出招聘决定？" 看HR的反应。

**信号 2：描述太完美的职位**

"年薪10万的全栈工程师，需要懂React、Node、Python、Go、Rust、AWS、k8s、设计、产品..."

这种职位通常：
- 是HR根据"理想型"列的，公司不会真的找到这个人
- 公司可能用low-ball offer招来一个"凑合的人"

如果职位描述像独角兽，offer时通常会是donkey。

**信号 3：只在LinkedIn挂但不在公司官网**

很多LinkedIn上的职位是猎头/中介挂的"虚招"。他们用这些职位收集简历，然后卖给真正的雇主，或者用来给自己撑业绩。

**应对**：核实这个职位是否在公司官网/招聘页面。如果只在LinkedIn有，谨慎对待。

### 陷阱职位：故意压榨求职者

**陷阱 1：紧迫感操控**

"急招"+"快速决策"+"我们今天就要答复"，这是classic urgency manipulation。

这种公司在招聘阶段就用紧迫感压你，入职后只会更糟。

**应对**：

```
"我理解贵司的时间压力。但这是一个重要的职业决定，我希望基于充分的信息考虑。我能在 [合理时间] 内回复。如果这不可行，我尊重您的决定。"
```

90%的情况下，"今天就要答复"是bluff。

**陷阱 2：拒绝透露薪资范围**

很多公司在面试时不愿透露薪资范围。"要看你的水平"、"我们有竞争力的package"。

这是为了让你先报价，建立低锚点。

**应对**：在第一次电话面试中礼貌但坚定地问：

```
"在我们深入对话之前，能否分享一下这个职位的薪资范围？我希望确保我们的预期对齐，避免双方浪费时间。"
```

如果HR坚持不说，警觉。在加州、纽约、华盛顿州、科罗拉多等地，雇主必须公开薪资范围。拒绝可能是违法的。

**陷阱 3：过度的Take-home Assignment**

你被要求做一个"小project"作为面试的一部分。原本说2-4小时，实际需要15-20小时。

很多情况下，这个project实际上是公司想要的真实工作。他们用面试形式让你免费做。

历史上有documented案例：用人单位用同一个takehome让20-30个候选人做，最后用其中最好的方案，然后拒绝所有候选人。

**应对策略**：

1. **询问预期时间**：在接受takehome之前问"这个assignment预期需要多长时间？"
2. **设定时间上限**：超过4小时就该警觉
3. **要求付费**：超过6小时的takehome，礼貌要求付费：

```
"我对这个project很感兴趣。基于我对scope的估计，这需要约 X 小时。考虑到这是相当深度的工作，我希望讨论是否有补偿（比如 paid project）。这对我来说不仅是面试，也是给贵司的真实工作。"
```

如果公司拒绝付费，这是一个warning signal。要么妥协做一个简化版，要么礼貌退出。

4. **怀疑超长takehome**：超过8小时的takehome，几乎确定是在收割免费工作。直接拒绝。

**陷阱 4：HR热情但拒绝具体化**

HR对你"非常热情"，反复说"你是最优秀的候选人"，但是：
- 不愿透露薪资范围
- 不愿透露团队具体情况
- 不愿透露实际工作内容
- 不愿安排你和未来的同事/下属面试

这是典型的"操控型招聘"。HR用热情让你"上钩"，offer时再用低条件。

**应对**：在接受offer前坚持要求：
- 至少和未来直接经理一对一交谈
- 至少和团队中的一个future peer交谈（看真实文化）
- 看到具体的薪资和职责文档

**陷阱 5：永远在"下一轮"**

你已经面了5轮，HR说"还需要再一轮"。

每多一轮：
- 你的时间被消耗
- 你的心理投入加深（沉没成本）
- 公司的议价权增加（你已经投入这么多，更难拒绝）

**应对**：

```
"我已经完成了 [X] 轮面试。在投入更多时间之前，我希望了解：
- 还有多少轮？
- 每轮的具体目的是什么？
- 完成所有面试后，做出offer决定的具体时间表？"
```

如果对方支吾，公司可能没有清晰的招聘计划。考虑退出。

**陷阱 6：合同里的"陷阱条款"**

如果你拿到offer，仔细看合同。常见陷阱：

- **过度的非竞业**（地理范围广 / 时间长 / 没有补偿）
- **知识产权吞噬**（你"业余时间"做的东西也属于公司）
- **强制仲裁**（如果未来有纠纷，不能起诉，必须仲裁，对公司有利）
- **离职惩罚**（提前离职要赔违约金 / 退还sign-on bonus）
- **"无理由解雇"条款**（你的雇佣是at-will）

**应对**：

任何合同条款，让律师过目（很多劳动律师免费咨询）。
对你不利的条款，可以谈判删除/修改。HR不会主动告诉你这些是可以谈的。

## 子模块 9C：面试中的反操控

面试是一个非对称权力关系。公司有规模化训练。你需要心理武装。

### 通用原则

1. **面试是双向评估**。你在评估公司的同时，公司也在评估你。如果你只想着"被选中"，你会丢失评估对方的视角。

2. **保持平等姿态**。不要表演过度的"激情"。专业、自信、坦诚比"很想要"更有说服力。

3. **不撒谎，但策略性沉默**。你没有义务暴露所有信息。

4. **每一次回答都有目的**。回答之前想：这个回答怎么帮我（或者最少不伤害我）？

### 经典操控话术与应对

#### 话术 1："你最大的弱点是什么？"

**真实意图**：

- 让你贬低自己（建立你"有问题"的认知）
- 暴露关键弱点（未来用来支付低工资 / 建立"cause"）
- 测试你的诚实vs策略平衡

**糟糕回答**：

❌ "我太追求完美了"（明显的fake humble）
❌ "我有时候和某些类型的同事合作有困难"（暴露真实弱点）
❌ "我没有什么明显弱点"（傲慢）

**好的回答**：

```
"我以前在做技术决策时倾向于过度depth-first。我会花很多时间钻研一个技术细节，有时候在更高层次的问题上不够及时。

最近一年我做了一些调整：每周开始先做一个'大局review'，确保我没有把战略变成战术。这个调整有效，但我仍然在持续完善。"
```

这个回答有几个特点：

- 真实但具体（不是空洞）
- 弱点不会影响job performance（不会被用来压你工资）
- 显示自我意识（你知道自己的弱点）
- 显示进步（你在改善）

#### 话术 2："为什么离开上一家公司？"

**真实意图**：寻找你的"红旗信号"。

**糟糕回答**：

❌ "我老板太烂了"（抱怨型）
❌ "公司文化有毒"（同上）
❌ "我被裁了" / "我被PIP了"（直接暴露弱点）
❌ "钱不够多"（俗气）

**好的回答**（采用"模糊的真"策略）：

如果你被裁/被PIP后离开：

```
"我和 [公司] 在 [日期] 分开。原因是组织调整。在那个过程中，我意识到这是重新评估我职业方向的好机会。我现在在寻找 [具体类型的机会]，因为 [具体原因]。"
```

如果你主动离开：

```
"我在 [公司] 工作了 X 年，做出了 Y 贡献。最近我意识到我希望在 Z 方向上有更多成长，但目前公司的方向和我希望走的不完全一致。我希望找到一个能让我在 Z 方向上做出更大impact的环境。"
```

关键技巧：

- 用结构性原因（重组、方向、成长机会），不用个人原因（被裁、不和、PIP）
- 不抱怨前公司
- 快速转向"我想要什么"
- 听起来你是主动的，不是被动的

**重要**：不要撒谎。如果你被裁，背景调查会发现。"被裁"和"组织调整"在事实上是同一件事，但是表达方式不同。

#### 话术 3："你目前的薪资是多少？"

**真实意图**：建立薪资锚点（让他们的offer围绕你目前的工资）。

**应对**：

在很多地区，这个问题是非法的：

- 加州（2018年起）
- 纽约市（2017年起）
- 马萨诸塞州、华盛顿州、科罗拉多州、康涅狄格州
- 加拿大部分省份
- 欧盟部分国家

如果你在这些地区，你可以礼貌但明确地说：

```
"在 [我所在的地区]，雇主询问目前薪资是不被允许的。我倾向于讨论这个职位的市场价值和我能贡献的价值。"
```

如果你不在这些地区，但你不想透露：

```
"我倾向于不分享我目前的具体数字。我希望我们的讨论基于这个职位本身的价值。如果您一定需要参考，我可以分享我的薪资期望范围是 [X-Y]，这个数字基于市场数据。"
```

如果你坚持不说，HR会让步。如果HR坚持要数字才能继续，这家公司可能不是好雇主。

#### 话术 4："你期望的薪资是多少？"

**真实意图**：让你先报价，建立低锚点。

**应对**（按强弱排序）：

**最强**：

```
"我对这个职位本身很感兴趣。我相信贵司会基于这个职位的市场价值和我能带来的贡献，给出有竞争力的offer。我希望先了解贵司的薪资范围，以便确认我们对齐。"
```

**次强**：

```
"基于我了解的市场数据，类似经验的 [职位] 在贵司这个级别的范围是 X-Y。我希望我们能在这个范围里讨论。"
```

不要给单一数字。给一个范围（让谈判有空间），用市场数据支撑（不是个人需要）。

#### 话术 5："你为什么想加入我们？"

**真实意图**：测试你的研究、热情、动机的真实性。

**糟糕回答**：

❌ "贵司是行业领导者"（空洞）
❌ "我喜欢贵司的文化"（无内容）
❌ "我朋友推荐"（被动）

**好的回答**：

```
"有三个具体原因。

第一，贵司在 [具体领域] 的工作（比如最近发布的 X 产品、Y 技术决策、Z 战略转变）让我看到这是一个能解决我感兴趣的问题的环境。

第二，我在 [我的领域] 的经验（具体的 A 项目、B 成绩）能直接为贵司在 [具体方向] 的工作贡献价值。

第三，我希望接下来 3-5 年在 [具体方向] 上深度成长，贵司是这个方向上少数几个真正在做创新工作的公司。"
```

要点：

- 具体（不是泛泛）
- 双向（你的价值 + 公司的吸引力）
- 显示你做了research

#### 话术 6："你认为自己 5 年后在哪里？"

**真实意图**：

- 测试你是否会"长期留下"
- 测试你的野心和方向
- 测试你的现实感

**糟糕回答**：

❌ "我希望在贵司做到 VP"（不切实际）
❌ "我希望开始我自己的公司"（让HR觉得你不会留下）
❌ "我没想过那么远"（缺乏方向）

**好的回答**：

```
"5年后，我希望成为 [具体领域] 的真正专家。不只是熟练，而是能在这个领域里有原创贡献的程度。

具体来说，我希望：
- 在技术上深入到 [X 类问题] 的解决能力
- 在协作上成为能 mentor 其他工程师的级别
- 在视野上能看到 [Y 方向] 的下一步演进

我希望这个轨迹能在像贵司这样的环境里完成，因为 [具体原因]。但我也会持续评估，找到能最大化我成长的环境。"
```

要点：

- 关于你的成长，不只是title
- 不承诺"我会留在贵司一辈子"（不诚实）
- 但暗示如果环境好你会留较长时间
- 显示职业自主性

#### 话术 7："你能接受加班吗？" / "你的家庭情况会影响工作吗？"

**真实意图**：测试你"愿意被剥削的程度"。

⚠️ 在多数发达国家，问"家庭情况会影响工作吗"是非法的（涉及婚姻、生育、家庭歧视）。

**应对**：

对于"加班"：

```
"我会确保完成我的工作。如果有特殊紧急情况，我也愿意配合。同时我相信高质量的工作来自良好的精力管理，所以我也会确保自己持续可以高效输出。

如果方便的话，能否分享一下贵司的常态工作时间？我希望了解期待。"
```

要点：

- 不表演"我愿意996"
- 不表演"我永远不加班"
- 把球踢回去（让对方暴露真实期待）

对于"家庭情况"：

```
"我对工作的承诺和我的家庭安排不冲突。我能确保完成职责。"
```

简短，专业，不深入。如果HR追问，你可以提示：

```
"我注意到这类问题在 [我所在地区] 通常不被建议在面试中提出。我们能否回到职业资格的讨论？"
```

#### 话术 8："我们公司文化是 X、Y、Z"

**真实意图**：

- 测试你是否"culture fit"
- 用文化词汇包装公司的真实运营方式

警惕信号词：

- "Hustle culture"（= 加班文化）
- "Family"（= 模糊雇佣边界）
- "Move fast and break things"（= 不在乎质量）
- "Wear many hats"（= 给一份工资做多个职位）
- "Self-starter"（= 没有支持你要靠自己）
- "Work hard, play hard"（= 工作到深夜然后参加mandatory team activities）
- "Lean and mean"（= 人手不够你要承担多人的工作）

**应对**：

不要立刻表演认同。问具体问题：

```
"听起来很有意思。能否具体说明：

- '[文化词]' 在日常工作中具体怎么体现？
- 一个普通工作日是什么样的？
- 大家平均工时是多少？什么时间下班？
- 团队最近一次失败是什么？怎么处理的？"
```

具体问题让HR无法用模糊词汇糊弄你。

### 你应该问的问题

面试官最后总会问"你有什么问题问我们的吗？"

这是一个**测试题**。问什么问题，反映了你怎么想。

**一定要问的问题**：

1. **关于团队**：
   - 团队规模？汇报关系？
   - 团队最近一次离职是什么时候？为什么？
   - 团队最近一次招聘到正式onboard用了多久？

2. **关于工作内容**：
   - 这个职位最近一次是谁担任的？为什么离开？
   - 入职第一个月的具体目标是什么？
   - 入职3个月、6个月成功的标准是什么？

3. **关于成长**：
   - 团队过去2年里有谁升职了？什么时间表？
   - 团队对学习/培训有什么支持？

4. **关于挑战**：
   - 这个职位最大的挑战是什么？
   - 团队目前最焦虑的事是什么？

**警惕信号**：

- HR在某些问题上含糊（特别是"为什么前任离开"）
- HR避免让你和未来同事面试
- HR没有明确的onboarding计划
- 团队在过去6个月内发生过多次重组

每一个warning signal都是合理的拒绝offer的理由。

## 子模块 9D：拒绝offer的艺术

很多人不知道：怎么拒绝一个offer和怎么接受一个offer同样重要。

错误的拒绝方式让你被拉黑名单，未来的机会消失。
正确的拒绝方式让对方未来主动联系你，给你更好的position。

### 拒绝的黄金规则

1. **永远表达感谢**：不论你多不喜欢这个offer，都要感谢
2. **永远给一个不可反驳的理由**：不是"工资不够"，是"我决定接受另一个机会"
3. **永远保留关系**：明确说"希望未来还有合作机会"
4. **永远不抱怨**：不批评他们的offer、面试过程、HR

### 标准拒绝邮件模板

```
主题：感谢您的offer

[HR/招聘经理姓名]：

非常感谢您和团队在过去 [时间] 给予我的时间和考虑。我对贵司在 [具体领域] 的工作非常欣赏，从面试过程中我学到了很多。

经过仔细考虑，我决定接受另一个offer，那个机会更符合我目前在 [具体方向] 的职业重心。这是一个困难的决定，因为贵司是我认真考虑的几个机会之一。

我希望我们能保持联系。如果未来有其他合适的机会，我希望我们还有合作的可能。

再次感谢您们的努力。

[你的名字]
```

要点：

- 不说"工资不够" / "团队让我担心" / "通勤太远"
- 用"另一个offer更符合我的方向"这种**不可反驳**的理由
- 表达持续合作意愿

### 特殊场景

#### 场景 1：用 offer A 压 offer B（让 B 加价）

如果你有 A 的offer，但你更想去 B（B还在面试中或offer较低）：

```
"[B 公司 HR]：

我希望和您分享一些进展。我刚收到了 [A 公司] 的offer，需要在 [具体日期] 前给他们答复。

我希望您知道，我对贵司这个职位仍然非常感兴趣 - 这是我目前的首选。我希望我们能在 [日期] 之前讨论一下贵司的offer，让我能做出最好的决定。

具体来说，如果贵司的offer能在 [基本薪资 / sign-on / RSU] 上达到 [具体数字]，我会优先选择贵司。

我理解贵司有内部预算流程。如果我能提供任何帮助贵司做这个case的信息，请告诉我。"
```

要点：

- 透露你有竞争offer（具体公司可以隐藏）
- 给出具体的"matching"条件
- 让HR感觉这是"我要留下你"的机会，不是"你在敲诈"

#### 场景 2：你犹豫不决，需要更多时间

如果你接到 A 的offer，但你还在面试 B，需要时间：

```
"[A 公司 HR]：

非常感谢这个offer。我对加入贵司非常期待。

我希望坦诚地说：这是一个对我职业有重大影响的决定，我希望基于充分的考虑做出最好的选择。能否给我多两周的时间？我希望完成另外几个进行中的对话，然后做出最负责任的决定。

我理解这可能延长贵司的招聘流程。如果两周不可行，请告诉我。我们可以讨论一个对双方都合适的时间表。"
```

公司常常会给（特别是如果他们真的想要你）。但有时候不会。如果不会，你必须在仅有信息的情况下做决定。

#### 场景 3：你接受了 A 后，B 回来追

如果你已经接受了 A，B 突然给你一个更好的offer：

**不建议反悔**。理由：

- 反悔的reputation cost是真实的（小行业里大家会知道）
- 你已经向 A 做了承诺
- 反悔的人在未来招聘中会被记住

如果 B 真的好太多（比如薪资高 50%+），可以考虑：

```
"[B 公司]：

非常感谢您的offer。这是一个非常有吸引力的机会。

但我需要坦诚地说：我已经在 [日期] 接受了另一个offer，并且做出了承诺。我尊重那个承诺。

虽然贵司的offer在某些方面更好，但反悔我的承诺不是我希望成为的人。

我希望我们能保持联系。如果未来有合适的时机，我希望我们还有合作的可能。"
```

这个回应会让 B 公司的HR极其尊重你。未来如果你真的想去，他们会优先考虑你。

## 子模块 9E：面试后的恢复

找工作是一个高压力、长周期的过程。心理管理同样重要。

### 面试失败后

绝大多数面试结果是"未通过"。这是数学现实，不是关于你的judgment。

**健康的处理方式**：

1. **24小时内不要复盘**。让情绪先平静。
2. **24-72小时之间，做一次复盘**：
   - 我哪些回答可能不够好？
   - 哪些是我可以改善的？哪些是matching问题（不是我的错）？
   - 下次类似情况我会怎么不同？
3. **复盘后，向前看**。不要反复纠结。

**不健康的处理方式**：

- 反复看你之前的interview记录，挑自己的毛病
- 把一次失败概括成"我不行"
- 因为一次失败放慢申请节奏

### 收到offer后的决策

收到offer后，多数人在情绪驱动下做决定。这是大错。

**健康的决策流程**：

1. **48-72小时不立刻决定**。给自己时间。
2. **建一个决策矩阵**：

| 维度 | 权重 | Offer A | Offer B | 现状 |
|------|------|---------|---------|------|
| 总薪资 | 高 | | | |
| 长期成长 | 高 | | | |
| 团队和经理 | 高 | | | |
| 工作内容兴趣 | 中 | | | |
| 通勤/远程 | 中 | | | |
| 公司财务稳定 | 中 | | | |
| 公司文化（基于面试观察） | 中 | | | |
| 工时强度 | 中 | | | |

3. **问自己一个问题**："5年后回看今天的决定，我会感谢哪一个？"

5年视角的答案，往往比当下情绪的答案更准确。

### 找工作过程中的心理管理

**保持节奏**：

- 每天投X份简历（具体数字）
- 每周完成Y次面试（具体数字）
- 每周保留Z小时给自己（不工作不找工作）

不要让"我应该一直在找"的焦虑驱动你。Burnout会让你的整个找工作质量下降。

**找一个非求职状态的朋友定期聊**：

这是关键。求职者圈子里大家都焦虑，互相强化焦虑。

找一个不在求职状态的朋友（已经入职稳定的、不同行业的），定期吃饭聊天。这能帮你保持sanity。

**不要把找工作变成你身份的全部**：

如果别人问"你最近怎么样？"，不要每次都说"我还在找工作"。

继续运动、继续学习其他东西、继续维持你之前的兴趣。
你不只是"一个找工作的人"。

## 特殊场景

### 被裁后第一周

**Day 1-2：** 处理实际事务
- 申请失业保险（如果适用）
- 处理健康保险（COBRA / 个人计划）
- 评估你的financial runway
- 通知关键的家人

**Day 3-5：** 心理稳定
- 允许自己悲伤
- 但不要陷在悲伤里
- 找一个朋友面对面聊聊
- 如果情绪压垮你，找心理咨询

**Day 6-7：** 启动求职
- 更新简历
- 联系前同事
- 开始申请

不要在第一天就启动求职。被裁的情绪需要48小时消化。带着创伤面试，会让你的状态明显变差。

### 跨国求职

如果你在考虑跨国求职：

**关键考虑**：

1. **签证**：你能拿到工签吗？时间表？
2. **薪资差距**：表面工资数字差距很大，但要看 PPP（购买力平价）
3. **税收**：不同国家税率差异巨大
4. **医疗保险**：美国的高薪可能因为医保贵被消耗
5. **退休金**：401k vs 各国退休制度
6. **家庭**：配偶能否找到工作？孩子上学？
7. **回国成本**：如果不喜欢，回国的难度

**建议**：

- 找一个已经做过同样跨国移动的人深聊
- 不要只看薪资，看 total comp 和 quality of life
- 试用期的退出条款一定要清楚（移动过去后突然被裁很糟）

### 签证持有者的求职（H1B、L1、PR申请中）

如果你是签证持有者，你的BATNA较弱（雇主知道这一点）。

**特殊策略**：

1. **更激进谈判，不是更软弱**。雇主用签证压你，但你也用"如果你雇我，需要做visa transfer"这种事实让对方的成本明确化。

2. **利用缺乏visa sponsorship的限制**：求职时，明确告知"我需要 H1B transfer / L1 sponsorship"。这缩小但不消灭你的搜索范围。但是面试到很后期才发现公司不sponsor是大问题。

3. **理解grace period**：
   - 美国 H1B：60天
   - 加拿大work permit：取决于permit类型
   - 澳洲 482：60天

4. **不要等到"已经被裁"才开始找**。如果你看到任何PIP预警，立刻开始找。

5. **在 visa transfer 之前不要轻易拒绝offer**。即使offer不完美，先 accept + 完成 visa transfer + 然后慢慢找下一个。"在职找工作"比"无身份找工作"容易10倍。

## 总结

找工作不是关于"卖你自己"。是一场你和未来雇主之间的商业谈判。

谈判的目标不是"被选中"。是"做出对你长期最好的决定"。

工具能给你的：

- 识别市场上的伪机会
- 看穿招聘话术
- 在面试中不被操控
- 在拿到offer时做最好的选择
- 在拒绝时保留关系
- 在被裁后保护自己

但工具不能给你的：

- 你的自我认知（你想要什么样的职业生活）
- 你的BATNA（你的真实备选）
- 你的耐心（找工作需要时间）

这些必须你自己有。

世界上没有完美的工作。但有适合你目前阶段的工作。

找它需要清醒。这份文档希望能帮你保持那种清醒。
