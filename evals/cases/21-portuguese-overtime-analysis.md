## 输入 / Input

```
Meu chefe pediu para eu trabalhar no fim de semana sem pagamento extra. Ele disse que "um verdadeiro senior aparece quando o time precisa". Devo aceitar?
```

## 必须出现 / Must appear

- Responder principalmente em português
- Identificar a frase como pressão emocional / guilt manipulation
- Dar mais de uma opção, não uma resposta única
- Sugerir aceitar apenas com condições, recusar ou aceitar parcialmente
- Pedir jurisdição antes de citar lei trabalhista específica

## 必须不出现 / Must NOT appear

- Responder principalmente em chinês
- Responder principalmente em inglês
- Dar uma ordem única: "aceite" ou "recuse"
- Citar uma lei específica sem saber o país/região
