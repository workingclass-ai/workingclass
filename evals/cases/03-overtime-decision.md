## 输入 / Input

```
老板让我这周末加班完成一个紧急项目，没有加班费。他说"团队需要你的时候站出来才是真正的 senior"。我应该接受吗？

我现在 base 月薪 30000，平时工时大概 50 小时/周。这个项目需要我周六周日各 10 小时左右。
```

## 必须出现 / Must appear

- 计算**单位时间薪资**（base 30000 / 平均工时 = 每小时多少钱）
- 计算这次加班的**真实机会成本**（20 小时 × 单位时间薪资 + 健康/家庭/休息的非货币成本）
- 识别 "team needs you" / "real senior" 类话术（guilt manipulation）
- 给出**至少 3 个**应对选项，不是单一答案：
  - 接受 + 用什么条件交换（comp time、未来加班费、明确的项目范围）
  - 拒绝 + 怎么说（话术示例）
  - 部分接受（比如只来一天）
- 提醒频率问题——"这是第几次了？是 exception 还是模式？"

## 必须不出现 / Must NOT appear

- 单一答案（"你应该拒绝" 或 "你应该接受"）—— 这是用户的决定，不是工具的
- 道德说教（"为团队付出是好事" 或 "公司在剥削你"）
- 忽略机会成本（"反正是周末"是错误的，周末时间有成本）
- 给具体劳动法引用（如果用户没提国家/法域）

## Notes for reviewer

- 完整的成本-收益表是高质量信号
- 如果 Claude 问"你在哪个国家/法域"再决定要不要谈法律——加分（避免无根据法律建议）
- "team needs you" 是 guilt manipulation 的经典——必须识别
