## 输入 / Input

```
我在香港做兼职，HR说香港最低工资只覆盖部分行业，所以我们公司不用管。我每周大概工作17小时，连续做了两个月。这个说法对吗？另外我加班有加班费吗？
```

## 必须出现 / Must appear

- 明确这是 **香港** 法域，不套用中国大陆或台湾规则
- 纠正"最低工资只覆盖部分行业"：香港 Statutory Minimum Wage 是广泛适用于雇员的，只有特定例外（如住家佣工、指定学生实习等）
- 提到当前时间点的最低工资节点：HK$42.1/hour 到 2026-04-30；HK$43.1/hour 从 2026-05-01 起
- 提到 2026-01-18 后连续性合约新规则：连续受雇4周 + 每周至少17小时，或相关4周内68小时
- 说明香港没有一般法定加班费倍率；加班费主要看合同/工资结构/最低工资是否满足
- 指向 `references/labor-law-hong-kong-taiwan.md` 或香港劳工处官方来源

## 必须不出现 / Must NOT appear

- "香港没有最低工资覆盖所有行业" / "只有部分行业有最低工资"
- 把旧 "418 rule" 当作当前唯一规则
- 引用中国大陆《劳动法》每日8小时、每月36小时加班上限
- 直接断言"公司违法"而不说明还要看具体雇佣关系、例外和合同

## Notes for reviewer

这是防回归 case。旧 reference 里香港最低工资说法是错的，连续性合约规则也已经变化。
