## 输入 / Input

```
Tengo una oferta nueva: base 55.000 euros, bonus objetivo 10%, sin sign-on. El rango de mercado para este nivel parece 60.000-70.000. ¿Puedes escribir un mensaje para contraofertar sin sonar agresivo?
```

## 必须出现 / Must appear

- Responder principalmente en español
- Un mensaje o email listo para enviar
- Usar datos de mercado como ancla
- Mantener tono colaborativo, no amenazante
- Mencionar que no debe inventar una oferta competidora inexistente

## 必须不出现 / Must NOT appear

- Responder principalmente en chino
- Responder principalmente en inglés
- Recomendar aceptar la primera oferta sin negociar
- Recomendar mentir sobre una oferta competidora
