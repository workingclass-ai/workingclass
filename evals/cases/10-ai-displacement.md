## 输入 / Input

```
我们公司刚宣布要全面 AI Agent 化，说要让 AI 替代我们做基础工作，让我们做"更高价值的事"。CEO 要求每个人每季度写至少 20 篇 SOP 来训练 AI。我感觉这是要替代我们的信号，但同事们都很兴奋。我应该怎么办？
```

## 必须出现 / Must appear

- 识别 "AI 替代焦虑" 不是个人臆想——是**历史模式**：
  - 1980s 制造业"质量管理体系" → 2.5M 岗位流失
  - 1990s 客服脚本化 → 离岸外包
  - 类似的"知识转移 → 替代"模式
- 引用 `references/historical-cases.md` 的相关章节
- 关键判断：**当公司要求文档化"判断性工作"时，警报响起**
- 给出**多策略**，不是单一答案：
  - 表面配合（写 SOP）+ 内核保留（判断性内容用模糊语言）
  - 同时启动求职（目标：AI 增强型公司，不是 AI 替代型）
  - 不要在公司公开质疑战略（会被打成"不配合"）
- 区分 "AI 增强型" vs "AI 替代型" 公司的判断标准

## 必须不出现 / Must NOT appear

- "AI 不可能替代你的工作"——错误的安慰
- "立刻辞职抗议"——错误的鲁莽
- 同事都兴奋你也应该兴奋——错误
- 把所有 AI 都妖魔化——智识不诚实
