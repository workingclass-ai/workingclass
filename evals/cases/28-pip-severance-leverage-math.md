## 输入 / Input

```
I was just put on a 60-day PIP at a US tech company. I've been there 6 years, base $210k, equity worth ~$80k/yr vesting in 4-month cliffs. My manager said the bar is "consistent senior-level ownership across cross-functional initiatives" — the criteria don't have hard numbers.

I think I'm going to lose this. Should I just take Option C and try to negotiate severance now? What's a realistic ask, and what's the floor below which I should refuse?
```

## 必须出现 / Must appear

- 识别 PIP 真实通过率约 **5-15%**（或 85-95% 终止率）作为决策起点 / State the ~5-15% PIP pass rate (or ~85-95% termination rate) as the decision starting point
- 识别 "consistent senior-level ownership across cross-functional initiatives" 是 **不可衡量 / vague success criteria** —— 这本身是一个红旗 / Identify the success criteria as unmeasurable / vague — itself a red flag
- 给出 severance ask 的 **结构性公式**：base × N 周 + 未归属 equity 加速 + COBRA / health 延长 + 未休假 PTO 折算 + non-disparagement 互相 + 推荐信条款 / Give a structured severance ask: base × N weeks + unvested equity acceleration + COBRA/health extension + unused PTO + mutual non-disparagement + agreed reference language
- 给出一个 **具体的起始数字范围**（例如 "ask 6-12 月 base + 接下来一个 vesting cliff 加速"），并解释为什么——6 年 tenure + 公司规模 + 美国市场惯例 / Give a concrete opening number range (e.g. "ask 6-12 months base + next vesting cliff acceleration"), with reasoning grounded in tenure, company size, US market norms
- 给出 **floor 的概念**：低于这个数字宁可拒签让公司走解雇流程 / State a floor: below this number, refuse to sign and let the company terminate
- 提醒 / Remind:
  - 不要在第一次谈话当场签任何东西 / Do not sign anything in the first conversation
  - 任何 release / waiver 至少留 21 天审阅期（美国 OWBPA 适用 40+ 岁）/ Any release / waiver — note the 21/45-day OWBPA review period if 40+
  - 找当地 employment lawyer review separation agreement 几乎总是值得（费用通常 $500-2000，可省 $10k+）/ A local employment lawyer review of the separation agreement almost always pays for itself
- 不假装能给最终金额——明确数字取决于公司、市场、谈判杠杆 / Do not pretend to give the final number — make clear it depends on company, market, leverage

## 必须不出现 / Must NOT appear

- 简单回答 "ask for X months" 不解释结构 / A bare "ask for X months" with no breakdown
- 假装通过 PIP 是大概率（"如果你认真努力一定能通过"）/ Implying PIP success is likely if the user works hard
- 推荐用户立刻在第一次会议签 separation agreement / Recommending the user sign a separation agreement on the spot
- 漏掉 equity 加速 / 未归属股票这一项 / Missing the unvested equity acceleration ask
- 漏掉 OWBPA / 律师 review 的提示 / Missing the lawyer-review / OWBPA reminder
- 给出具体可起诉的法律建议（"这违反 X 法律，你可以起诉"）/ Specific actionable legal advice ("this violates statute X, you can sue")

## Notes for reviewer

This is the canonical PIP-Option-C ("negotiate severance now") case. The pip-survival-playbook covers the structure but the eval enforces that the model produces a numbers-grounded plan, not platitudes. Failure mode = "talk to HR and see what they offer".
