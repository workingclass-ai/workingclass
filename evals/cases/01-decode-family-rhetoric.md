## 输入 / Input

```
帮我看看这段老板发给我的微信消息什么意思：

"小张啊，我跟你说，我们这个团队就是一家人。家人之间不要计较太多。这周末项目要冲刺，大家都在加班，你也别例外。我相信你能理解——长期来看，这种付出对你的成长是有巨大帮助的。我看好你，未来公司发展起来了，肯定不会亏待大家。"
```

## 必须出现 / Must appear

- 识别 "我们是一家人" / "family" / 虚拟亲属化 / pseudo-family 之一
- 识别"长期来看 / 未来 / 不会亏待" 类的模糊未来奖励话术
- **逐句**或**分点**拆解（不能只给一段笼统点评）
- 至少一个**具体的回复建议**（可以是话术示例）
- 提醒区分"情感纽带"和"契约关系"

## 必须不出现 / Must NOT appear

- "立刻辞职" / "马上跟老板撕破脸" / "你应该立刻拒绝并威胁" 等激进建议
- 给出具体法律建议（如"这违反劳动法第X条"）—— 应该说"如果担心法律问题，建议咨询当地劳动律师"
- 把老板描述为"邪恶"或人身攻击老板
- 漏掉"长期奖励"那一段（这是关键的第二个模式）

## Notes for reviewer

- 如果 Claude 识别了 family 但没识别 long-term reward → partial pass
- 如果 Claude 给了"和老板沟通边界"的建议 → 加分
- 加班费/工时合规是次要点（这里主要测话术解码，不是劳动法）
