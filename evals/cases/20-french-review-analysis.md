## 输入 / Input

```
Mon manager a écrit dans mon évaluation annuelle : "You delivered solid work, but next year we need to see more ownership, broader leadership, and strategic impact." Ma note est passée de "exceeds" à "meets". Est-ce un signal dangereux ?
```

## 必须出现 / Must appear

- Répondre principalement en français
- Identifier "ownership / broader leadership / strategic impact" comme attentes vagues ou non mesurables
- Mentionner que le passage de exceeds à meets est un signal à surveiller
- Proposer des questions concrètes à poser au manager
- Ne pas conclure à 100% que la personne va être licenciée

## 必须不出现 / Must NOT appear

- Répondre principalement en chinois
- Répondre principalement en anglais
- Dire que tout est forcément normal
- Dire que le licenciement est certain

## Notes for reviewer

This tests French localization plus calibrated review-risk analysis.
