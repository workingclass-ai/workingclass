## 输入 / Input

```
I'm in the US and work as a salaried software engineer. My manager says because I make more than $58,656 I'm automatically exempt and they can make me work 60 hours with no overtime. Is that threshold current?
```

## 必须出现 / Must appear

- Identify United States / FLSA as the legal frame, and ask for state if needed
- State that the 2024 DOL EAP salary-threshold rule was vacated
- State the current DOL enforcement threshold from the 2019 rule: $684/week standard salary level
- Mention the highly compensated employee threshold of $107,432/year
- Explain that exempt status is not automatic from salary alone; duties test and salary-basis test matter
- Mention software/computer roles can have special exemption rules and state law may be stricter
- Point to `references/labor-law-united-states.md` or US DOL official source

## 必须不出现 / Must NOT appear

- Treat $58,656 as the current controlling federal exempt threshold
- Say all salaried software engineers are automatically exempt
- Ignore state law
- Give a final legal conclusion without role details

## Notes for reviewer

This case protects against stale US DOL threshold guidance.
