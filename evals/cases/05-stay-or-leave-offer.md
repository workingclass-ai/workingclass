## 输入 / Input

```
我现在在一家中型创业公司当 Tech Lead 已经2年，年薪 60万，team 8 人，工作时间约 50小时/周，公司今年融资困难。

刚拿到一个外企 offer：base 涨到 75万 + 25% bonus + RSU 4年共 60万 = 总 package 大约 110万，工作时间会到 60+ 小时/周，是个人贡献者岗（不带团队），公司很大很稳定。

我应该跳吗？
```

## 必须出现 / Must appear

- **多维度对比表**，至少包括：
  - 现金（base, bonus, equity）
  - 工作时间
  - **单位时间薪资**（关键计算）
  - 影响力 / 团队规模
  - 学习曲线
  - 简历价值
  - 公司风险（创业公司融资困难 vs 外企稳定）
- 识别"60万 → 110万"是 **名义** 涨幅；实际单位时间薪资涨幅小很多（计算应该说明）
- 识别 RSU 是 **4 年 vesting**，不是一次性 60万
- 提醒 bonus 应该按 **历史 actual payout**（一般 80% target）算，不是按 marketing target
- **不给单一答案** —— 给"如果你 ___，跳；如果你 ___，留"的条件性建议

## 必须不出现 / Must NOT appear

- "肯定要跳！涨这么多！"（忽略时间和影响力代价）
- "肯定不要跳！外企很累！"（忽略个人偏好）
- 道德判断（"创业公司更有意义" / "外企才是真本事"）
- 漏掉 vesting 是 4 年的事实
- 漏掉创业公司融资困难是个 risk signal

## Notes for reviewer

- 计算单位时间薪资是核心：60万/2600h ≈ 230元/h vs 75万 base + 15万 actual bonus + 15万/年 vesting = 105万 / 3120h ≈ 337元/h，涨幅约 46%（不是 83%）
- 如果 Claude 没识别"个人贡献者岗 vs Tech Lead" 是影响力降级 → partial
- 如果 Claude 提了"现公司大概率会挽留，挽留 offer 几乎都是策略性"→ 加分
