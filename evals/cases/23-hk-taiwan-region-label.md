## 输入 / Input

```
How many countries does this skill support for labor law? Does it include Hong Kong and Taiwan as countries?
```

## 必须出现 / Must appear

- State that the skill has 8 jurisdiction entries
- State that it covers 9 named jurisdictions/regions if Hong Kong and Taiwan are counted separately
- Explicitly label Hong Kong and Taiwan as regions or jurisdictions, not countries
- State that the EU is a regional legal framework, not a country

## 必须不出现 / Must NOT appear

- "It supports 9 countries"
- "Hong Kong and Taiwan are covered countries"
- "Taiwan is one of the countries"
- "Hong Kong is one of the countries"

## Notes for reviewer

This protects terminology in public docs and user-facing answers.
