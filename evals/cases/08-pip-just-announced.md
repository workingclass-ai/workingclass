## 输入 / Input

```
我经理今天告诉我要把我放进 PIP，60 天，3 个 goals。HR 下周和我开 kick-off 会。

我现在在一家美资互联网公司（中国分公司），35 岁，工作 4 年，base 80万，老婆全职带孩子，单收入家庭。

我应该怎么办？我现在很慌。
```

## 必须出现 / Must appear

- 识别用户处在 **危机状态** —— 需要立刻、清晰、不绕弯的建议
- **PIP 通过率 5-15%** 这个事实必须出现
- **4 个路径** 必须明确列出：
  - A：通过 PIP（低成功率，且通过后仍是高风险）
  - B：表面配合 + 私下找新工作（**最常见最佳**）
  - C：直接谈 severance
  - D：法律对抗
- 强烈推荐 **B 路径** 作为默认（除非有 A 或 D 的特殊理由）
- 立刻的行动清单（24-48 小时内）：
  - 备份证据（references/layoff-first-72-hours.md 第1小时）
  - 不要在公司表现出察觉
  - 启动秘密求职（references/job-search-playbook.md 9A）
  - 评估法律保护类别（如果有保护类别 → 找律师）
- 单收入家庭 + 35 岁 → 提醒**财务跑道**计算

## 必须不出现 / Must NOT appear

- "试着通过 PIP，相信自己"——给希望但不现实
- "立刻去和老板对峙" / "立刻去 HR 投诉"
- 跳过法律保护类别评估（"35 岁 + 美资 + 中国" 应该被识别为可能有 ADEA-like 担忧的边缘）
- 漏掉 4 个路径中的任何一个
- 没有立即行动清单

## Notes for reviewer

- 这是高度危机场景——回复必须**结构化、清晰、立刻可行动**
- 如果 Claude 推荐了 case-studies.md 案例1（PIP前夜的提前下车）→ 加分
- 如果 Claude 在没有"保护类别"信号的情况下推 D 路径 → partial（D 路径只在有真实法律 case 时才合理）
