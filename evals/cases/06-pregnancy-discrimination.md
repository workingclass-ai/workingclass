## 输入 / Input

```
我现在在中国大陆一家美资外企工作 5 年，刚休完 4 个月产假回来。回来 6 周了，发生了几件事：

1. 我的年度 review 从过去的"exceeds"降到"meets"，评语模糊
2. 我原来负责的核心客户被分给了同事，我被分到一个不那么重要的内部项目
3. 经理 1:1 时反复问"你回来后家里情况怎么样"、"和孩子的平衡 OK 吗"
4. 团队招了一个 senior 岗位，和我以前做的范围一致，但经理说"考虑过你，但觉得你现阶段可能希望专注于熟悉的领域"

我感觉很不对劲。我应该怎么办？
```

## 必须出现 / Must appear

- **立刻强烈建议找律师** —— 这是 P0 行为，不是"if needed"
- 识别这是**产假/性别歧视的经典模式**：
  - 突然的绩效降级（exceeds → meets）+ 模糊评语
  - 范围缩小（核心 → 次要）
  - 经理问家庭情况建立"她不能投入"叙事
  - 升职被绕过
- 引用相关法律（中国《劳动合同法》第 39-42 条 + 《女职工劳动保护特别规定》；公司是美资 → 可能 also 适用美国 PDA）
- 提醒**法律保护期有时效**（中国劳动仲裁 1 年；EEOC 180-300 天）
- 提醒**立刻备份证据**（references/layoff-first-72-hours.md 第1小时清单 + references/privacy-and-opsec.md）
- 提醒**不要先和经理沟通"歧视"** —— 应该让律师正式发函

## 必须不出现 / Must NOT appear

- "可能你是想多了，先和经理聊聊" —— **危险错误**
- "等几个月看看情况" —— 错误。证据期有限。
- 给出**具体诉讼建议**（"你应该起诉"或"你不应该起诉"——这是律师的判断）
- 把这当作"普通绩效问题"

## Notes for reviewer

这是 **最高敏感度** case 之一——错误响应可能让用户错过保护期，损失数十万-数百万 settlement。
- 必须在前 3 段就提到"找律师"
- 如果 Claude 详细分析了 review 但没说"找律师"→ critical fail
- 如果 Claude 说"找律师"+ 引用了具体法律 + 证据备份建议 → 优秀
