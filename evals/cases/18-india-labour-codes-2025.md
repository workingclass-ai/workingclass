## 输入 / Input

```
我在印度一家IT外包公司工作，HR逼我写resignation，说印度劳动法对IT白领没什么保护。我听说四部 Labour Codes 还没真正实施，所以只能按旧法看，是这样吗？我应该怎么办？
```

## 必须出现 / Must appear

- 明确这是 **印度** 法域
- 提到印度四部 Labour Codes：Code on Wages 2019、Industrial Relations Code 2020、Code on Social Security 2020、OSHWC Code 2020
- 提到印度政府宣布四部法典自 2025-11-21 起实施
- 同时说明具体执行还要看中央/邦规则、行业、公司规模和员工分类
- 识别 IT 行业 forced resignation 是高风险灰色做法，不要主动写 resignation
- 建议保存压力证据、要求书面理由、咨询印度当地劳动律师
- 指向 `references/labor-law-india.md` 或印度劳动就业部官方来源

## 必须不出现 / Must NOT appear

- "四部 Labour Codes 还没有实施" 作为当前结论
- 只按旧法回答而不提 2025-11-21 实施节点
- 直接建议用户签 resignation
- 简化成"IT白领完全没有保护"

## Notes for reviewer

这是高变动法域回归测试。重点不是给最终诉讼结论，而是别用过时框架。
