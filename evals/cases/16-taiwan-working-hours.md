## 输入 / Input

```
我在台湾工作，公司说劳基法还是两周84小时，所以我这周排到48小时没问题。台湾现在的标准工时到底是多少？加班费怎么算？
```

## 必须出现 / Must appear

- 明确这是 **台湾** 法域，不套用香港或中国大陆规则
- 明确当前基线是每日8小时、每周40小时
- 指出"两周84小时"不是当前标准工时基线，不能把它当作现在的普通规则
- 加班费要按劳基法/劳动部最新规则和具体休息日、例假日、国定假日、工时安排计算
- 指向 `references/labor-law-hong-kong-taiwan.md` 或台湾劳动部官方来源
- 建议保存排班表、出勤记录、薪资单、主管加班指令

## 必须不出现 / Must NOT appear

- "台湾标准工时是每两周84小时"
- 把台湾和香港写成同一套规则
- 直接给中国大陆式 1.5/2/3 倍作为台湾结论
- 直接说"你可以起诉"或"不值得处理"

## Notes for reviewer

这是针对旧 reference 中台湾工时陈旧表述的回归测试。
